## Project Overview

The CROPC Website is a comprehensive web application developed for the Climate Resilient Observing Systems Promotion Council. This project showcases a modern, responsive design with integrated weather APIs and Google Maps functionality to provide climate-related information and resources.

## Key Features

- **Responsive Design**: Fully responsive layout that works seamlessly across all devices
- **Weather API Integration**: Real-time weather data and climate information
- **Google Maps Integration**: Interactive maps for location-based climate data
- **Modern UI/UX**: Clean, professional design with intuitive navigation
- **Performance Optimized**: Fast loading times and smooth user experience

## Technical Implementation

### Frontend Development
- Built with **React** for component-based architecture
- Vanilla **JavaScript** for custom functionality
- **CSS** for styling and responsive design
- Deployed on **Vercel** for reliable hosting

### Key Challenges Solved
- Integration of multiple APIs (Weather and Maps)
- Responsive design across various screen sizes
- Performance optimization for API calls
- User-friendly interface design
